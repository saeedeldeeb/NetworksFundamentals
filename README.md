# Networks Fundamentals

Interactive Angular educational platform teaching network engineering fundamentals. Built as a visual companion to Hussein Nasser's Udemy course.

## Course Reference

This project is based on **[Fundamentals of Network Engineering](https://www.udemy.com/course/fundamentals-of-networking-for-effective-backend-design/?couponCode=NET-FEB2026-L)** by [Hussein Nasser](https://www.husseinnasser.com/).

## Tech Stack

- Angular 21 (standalone components, signals, lazy routing)
- GSAP 3.14 with ScrollTrigger and MotionPathPlugin
- Vitest
- Prettier

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:4200](http://localhost:4200).

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Dev server at localhost:4200 |
| `npm run build` | Production build |
| `npm test` | Run unit tests (Vitest) |
| `npm run watch` | Dev build with file watching |
