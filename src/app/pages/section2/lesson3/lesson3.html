MAC Address (Media Access Control)

Every network card has a unique MAC address: 00:1A:2B:3C:4D:5E (6 bytes, 48 bits). This is the lowest level of addressing.

The Broadcast Problem

On a shared network, when Host A sends to Host B:

Everyone receives the frame. Only the intended recipient processes it — others drop it. This is how public Wi-Fi sniffing worked.

Why MAC Addresses Aren't Enough

No Routability

MAC addresses are random — no location info

No Segmentation

Cannot divide into smaller networks

Global Scanning

Finding a MAC globally means scanning everything

Key insight: MAC addresses are globally unique but provide no location information — like having a unique name but no address.
