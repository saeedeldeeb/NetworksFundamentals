Lesson 3

ICMP, Ping, and Traceroute

ICMP (Internet Control Message Protocol) is a critical Layer 3 protocol used for network diagnostics and error reporting.

What is ICMP?

Layer 3 (Network)
Uses IP directly, no ports
Protocol # 1
Listener None needed

Ping

Sends an ICMP Echo Request and waits for an Echo Reply. Tests if a host is reachable and measures round-trip time.

Why ICMP Gets Blocked

Some firewalls block ICMP because of flooding attacks, network probing by attackers, and backchannel attacks using ICMP for covert communication.

TCP Black Hole (ICMP Blocking Problem)

Key insight: ICMP is essential infrastructure. Blocking it can cause hard-to-debug network issues like TCP black holes. When ping doesn't work, don't assume the host is down — ICMP might be blocked.
